## 📌 專案核心簡介

本專案是 **北市大暑期地球科學研習課程** 的官方資源倉庫[cite: 1, 3]。本課程以**地震學與地球物理學**為主軸，同時導入 AI／Agent 工具與雲端平台，協助中小學現職教師將硬核的科學知識，轉化為可帶回學校使用的互動教材、網頁與教學專題[cite: 1, 3]。

### 🎯 4 大學習核心
1. **地科理論深化**：從基礎地震動觀測、震央定位到彈性波波形分析[cite: 1]。
2. **AI & 雲端工具實作**：熟悉 GitHub、Vercel、Hugging Face、Google GAS 與 Hermes agent 等工具鏈[cite: 1]。
3. **教學現場轉化**：將 IRIS 國際地震觀測資源轉化為中小學互動學習單[cite: 1]。
4. **數位作品產出**：每週完成一項可檢核指標，期末發表一套完整數位教案[cite: 1]。
